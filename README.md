# VisionX Robotics | 多模态AI视光机器人官网

基于 PPT 资料打造的科技炫酷风格项目展示网站，设计灵感来自 Apple、NVIDIA、SpaceX 等顶级科技公司。

## 快速启动

直接双击 `index.html` 在浏览器中打开，或使用本地服务器：

```bash
cd /Users/leo/vision-robot-website
python3 -m http.server 8080
# 访问 http://localhost:8080
```

或使用 Node.js：

```bash
npx serve .
```

## 网站结构

- **首页 Hero** - 主视觉与品牌主张
- **01 项目摘要** - 核心优势与市场规模数据
- **02 行业痛点** - 医疗端/工具端/服务端痛点
- **03 产品方案** - VisionScan / VisionDiagnose / VisionTreat
- **04 技术突破** - 专利、论文、全球落地
- **05 核心团队** - 创始人及顾问介绍
- **06 商业模式** - 盈利模式与 Pre-A 融资计划
- **07 联系我们** - 公司信息与使命

## 设计特点

- 深色赛博风格
